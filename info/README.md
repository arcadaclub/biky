## Biky – Info-Seite

Diese Seite ist eine kleine Landingpage (Marketing/Info) für Biky, getrennt von der eigentlichen App.

### Pfad

- `mitglieder/biketour/info/`

### Aufruf

- Im Browser: `.../mitglieder/biketour/info/`

### Technik

- **Parallax**: leichte CSS/JS-Parallax-Ebenen (ohne externe Libraries)
- **Animationen**: Scroll-Reveal via `IntersectionObserver`
- **A11y**: Skip-Link, semantische Sections, `prefers-reduced-motion`
